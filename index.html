eYRC 2022-23 Krishibot

Collaboratively programmed an Autonomous Ground Vehicle (AGV) for navigating a greenhouse environment to locate and pluck targeted yield. Successfully implemented a PD control system using LiDAR data for precise lane keeping and trajectory correction in both Gazebo simulation and remote hardware. Utilized OpenCV for image processing and centroid detection to estimate fruit positions, and utilized the Moveit motion planning framework to autonomously control the onboard UR5 manipulator for the plucking process.
