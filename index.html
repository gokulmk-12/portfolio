eYRC 2022-23 Krishibot

Collaboratively programmed an Autonomous Ground Vehicle (AGV) for navigating a greenhouse environment to locate and pluck targeted yield. Successfully implemented a PD control system using LiDAR data for precise lane keeping and trajectory correction in both Gazebo simulation and remote hardware. Utilized OpenCV for image processing and centroid detection to estimate fruit positions, and utilized the Moveit motion planning framework to autonomously control the onboard UR5 manipulator for the plucking process.

Exploring Grasping Algorithms

Researched the Point Cloud Library to develop a lightweight grasp detection method using Euclidean Clustering and K-D Trees from RGB-D camera data. Integrated grasping algorithms such as Graspnet, HAF, and GPD with the Industrial UR5 Manipulator and OnRobot 2-finger gripper via ROS, implementing them in both Gazebo simulation and real-world setups. Evaluated these algorithms based on accuracy, precision, estimation time, and grasp success rate through extensive experiments.

eYRC 2023-24 Geoguide

Participated in a 6-month long competition, developing an autonomous robot to navigate an arena using an overhead camera for guidance. Events detected by the camera with CNN were communicated to the robot via a web server. The robot, equipped with IR sensors and a differential drive system, used ArUco markers as GPS coordinates to navigate and efficiently visit event locations by selecting the shortest path.

Motion Planning KUKA Youbot

This project utilizes a mobile manipulator, specically the KUKA Youbot, to continuously trace a trajectory using Sampling Based Planning Algorithms (RRT*, BiRRT*) while minimizing end-effector pose deviation through optimal control.
